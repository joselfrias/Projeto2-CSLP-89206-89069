\doxysubsection{Detailed Description}
Estrutura para representar uma imagem em binário. 

Esta estrutura permite a representação de uma imagem em binário, que é caracterizada por um conjunto de pixeis \mbox{\hyperlink{structBIN__PIXEL}{B\+I\+N\+\_\+\+P\+I\+X\+EL}}, definidos anteriormente, pelo número de elementos da matriz binária, por um número de linhas e colunas e consequentemente por um tamanho.
