\doxysubsection{Detailed Description}
Estrura para representar uma Imagem R\+GB. 

Esta estrutura irá possibilitar a representação de uma imagem R\+GB, que será caracterizada por um conjunto de estruturas \mbox{\hyperlink{structRGB__PIXEL}{R\+G\+B\+\_\+\+P\+I\+X\+EL}}, pelo número de linhas e colunas, pelo seu número de elementos e por um tamanho.
