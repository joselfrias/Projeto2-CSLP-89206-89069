\doxysubsection{Detailed Description}
Estrutura para representar a componente grayscale existente num pixel de uma imagem Grayscale. 

Dada uma imagem Grayscale, cada pixel é influenciado só por uma componente, podendo assumir esta um valor entre 0 e 255, sendo então representada por um unsigned char, que também assume um conjunto de valores possíveis igual ao descrito anteriormente.
