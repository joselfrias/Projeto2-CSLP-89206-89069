\doxysubsection{Detailed Description}
Estrutura para representar a componente binária existente num pixel de uma imagem binária. 

Dada uma imagem binária, cada pixel só pode assumir um valor de dois possíveis\+: 0 ou 1. No entanto, foi usado um unsigned char para representar cada pixel, o que não se torna muito eficiente.
