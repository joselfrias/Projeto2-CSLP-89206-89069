Detailed Description

Estrutura para representar uma imagem em Grayscale.

Esta estrutura permite a representação de uma imagem em Grayscale, que é caracterizada por um conjunto de pixeis GRAY_PIXEL, definidos anteriormente, pelo número de elementos da matriz Grayscale, por um número de linhas e colunas e consequentemente por um tamanho.
