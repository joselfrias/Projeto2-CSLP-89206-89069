Detailed Description

Estrura para representar uma Imagem RGB.

Esta estrutura irá possibilitar a representação de uma imagem RGB, que será caracterizada por um conjunto de estruturas RGB_PIXEL, pelo número de linhas e colunas, pelo seu número de elementos e por um tamanho.
